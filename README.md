# ReminderApp

Reminder App is a free and quick app that enables users to set reminders for tasks using alarms and notifications. It is available offline.

It allows one to set reminders related to:
•	Exams
•	Prayer
•	Taking pills
•	Doing exercise
•	School
•	Jogging
•	Doing push-ups
•	Medicine
•	Kids
•	Drinking water
•	Developing a habit
•	Bible reading
•	Study
•	Eating
•	Standing up
•	Vaccination
You name it.
